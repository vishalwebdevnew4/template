# Swigo - Fast Food And Restaurant React Template

A modern React-based template for fast food and restaurant websites, built with React, Vite, Bootstrap 5.3.3, and jQuery 3.7.1.

## Features

- ✅ React 19.2.0 with Vite
- ✅ Bootstrap 5.3.3
- ✅ jQuery 3.7.1
- ✅ React Router DOM for navigation
- ✅ Fully Responsive Design
- ✅ Modern UI/UX Design
- ✅ Multiple Home Page Layouts
- ✅ 40+ Inner Pages
- ✅ Performance Optimized
- ✅ SEO Friendly

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Customization

### Colors
Edit CSS variables in `src/index.css`:
```css
:root {
  --primary-color: #ff6b35;
  --secondary-color: #1a1a1a;
  --text-color: #666;
  --heading-color: #1a1a1a;
}
```

### Content
All content is in component files. Edit the respective component files to customize text, images, and functionality.
